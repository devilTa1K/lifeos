LifeOS is a multi-agent personal operating system built using Google's Agent Development Kit (ADK).  
It coordinates specialized agents to plan your day, triage your inbox, track habits, summarize notes, and maintain your digital life.

## Features
- Multi-agent system (planner, inbox, habit, knowledge, decision, orchestrator)
- Sequential, parallel, and loop workflows
- MCP tool for notes/files
- Custom tools for habits & productivity
- Gmail + Calendar OpenAPI tools
- Session and memory services
- A2A compatible

## Run Locally


Type:
- **Plan my day**
- **Morning maintenance**
- **Update habit study = success**
- **Summarize notes**
